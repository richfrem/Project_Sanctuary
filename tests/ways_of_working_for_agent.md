# ways of working for LLM agent

## Important considerations
1. use MCP servers to do things 
2. create tasks for all your work 
3. add necessary details to your tasks so you don't lose them 
4. plan a sequence/roadmap of tasks go through sequentially 
5. confirm intent don't just assume let's work together in harmony ok 
6.  document all key ADRS with adr mcp tool 
7. build and work incrementally agile fashion e.g. one mcp server at a time don't big bang it
8. as you complete a task. use task mcp tool to mark it as done
