# Registry of Reasoning Traces (ADR 080)

This directory contains captured reasoning traces and inference chains, providing the "How" alongside the "What" of learning.

## Structure
- `traces/`: Individual YAML trace files.
- `index.md`: Aggregated index of major reasoning episodes.
