# Clean RLM Cache

Remove stale entries (deleted files) and orphans (files outside manifest scope).

## Usage
```bash
# Dry run — see what would be removed
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/cleanup_cache.py --type legacy

# Actually remove stale entries
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/cleanup_cache.py --type legacy --apply

# Also prune orphans (entries not matching manifest scope)
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/cleanup_cache.py --type tool --apply --prune-orphans

# Remove failed distillation entries
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/cleanup_cache.py --apply --prune-failed
```

## Safety
- **Dry run by default** — nothing is deleted without `--apply`
- Verbose mode with `--v`
